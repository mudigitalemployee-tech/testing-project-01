Executive Summary

This report presents the output of the Batch ETL Pipeline for Sales Analytics built on the Superstore Sales dataset. The pipeline ingested 10,000 transactions spanning 2021–2023, applied cleaning and transformation rules, loaded a star schema in PostgreSQL, and computed aggregated KPI summaries. All data quality checks passed. The pipeline is orchestrated via an Apache Airflow DAG on a daily schedule.
